## What this project is about
This game was made during Bootcamp on week 7 to use the techniques that we have learned in 6 weeks. This game use signup and login functionality to track the players. We use a token-based login system using JWT (JSON Web Tokens).

The game supports multiple simultaneous games with a lobby system. For the frontend of this project, check out [this](https://github.com/marloesverdonk/dice-game-client) repository.
